## Reviewed references

- shadcn/ui: https://github.com/shadcn-ui/ui
  - Relevant because this project already uses shadcn/Radix-style components, Tailwind utilities, and a local component-copying model.
- Radix Primitives: https://github.com/radix-ui/primitives
  - Relevant for accessible base interactions such as dialogs, tabs, sheets, popovers, sliders, and bottom-sheet style controls.
- Headless UI: https://github.com/tailwindlabs/headlessui
  - Relevant for unstyled accessible interaction primitives that fit Tailwind-heavy prototypes.
- React Native Paper: https://github.com/callstack/react-native-paper
  - Relevant for mature mobile information hierarchy, Material-style navigation, lists, surfaces, and touch targets.
- TDesign Mobile React: https://github.com/Tencent/tdesign-mobile-react
  - Relevant for Chinese mobile product conventions, compact controls, list density, and mobile component taxonomy.
